## General Image Guidelines

### Quality Requirements
- **Resolution**: Minimum 1200px width for landscape images
- **Format**: JPG or PNG
- **File Size**: Keep under 500KB per image for optimal loading
- **Color**: Vibrant, well-lit images

### Content Guidelines
- Show diverse pilgrims (men, women, elderly)
- Capture spiritual moments (prayer, Tawaf, dua)
- Include iconic Islamic architecture
- Golden hour lighting preferred
- Clear, sharp focus

### Where to Get Images
1. **Stock Photo Websites**:
   - Shutterstock (holy sites, Islamic themes)
   - Getty Images (Mecca, Medina)
   - Adobe Stock (pilgrimage images)
   - iStock (Islamic travel)

2. **Free Alternatives**:
   - Unsplash (search: Mecca, Medina, mosque)
   - Pexels (search: Islamic, pilgrimage)
   - Pixabay (search: Ka'bah, Hajj)

3. **Custom Photos**:
   - Hire professional photographer during Umroh/Haji season
   - Partner with travel agencies for image sharing
   - Use high-quality drone footage

## Adding New Images

1. Place images in: `/themes/lafatour/assets/images/`
2. Use in templates: `{{ 'assets/images/your-image.jpg'|theme }}`
3. For package images: Upload via Backend > CMS > Media Manager

## Image Optimization Tips

Before uploading:
1. Compress images using TinyPNG.com or ImageOptim
2. Resize to recommended dimensions
3. Use WebP format for better compression (optional)
4. Add descriptive alt text for accessibility
